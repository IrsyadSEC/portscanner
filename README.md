# portscanner
Port scanner 

# Overview
this portscanner is easy to use for beginner 

# Instalation & Usage
- git clone https://github.com/IrsyadSEC/portscanner.git
- cd portscanner
- python3 portscanner.py <ip>
- Ex:python3 portscanner.py 192.168.50.6
